description: The Watch-Command is a PowerShell semi-equivalent of the respective Linux Watch utility. The Watch-Command cmdlet runs a command repeatedly, displaying its output and errors. This allows you to watch the program output change over time. By default, command is ran every 2 seconds and watch will run until interrupted.
